#  Croma Electronics Data Analysis using SQL

##  Project Overview
This project focuses on analyzing product inventory data for Croma – The Electronics Megastore using SQL.
The goal is to extract meaningful insights related to product pricing, manufacturer performance, and inventory management.

##  Tools & Technologies
- SQL
- Relational Database Concepts

##  Dataset Description
The project uses two tables:

1. Product Inventory
   - Product_Code, Product_Name, Product_Price, Manufacturer_Code  

2. Manufacturers
   - Manufacturer_Code, Manufacturer_Name  

Both tables are connected using the **Manufacturer_Code** key.

##  Key Analysis Performed
- Retrieved product details and pricing information  
- Applied filtering using conditions (WHERE, BETWEEN, LIKE)  
- Used aliases and string operations  
- Performed aggregate functions (AVG, SUM, COUNT)  
- Joined tables to combine product and manufacturer data  
- Identified cheapest and most expensive products  
- Analyzed manufacturer-wise product pricing  
- Performed sorting and data manipulation  
- Applied updates and discount operations  

##  Key Insights
- Identified products within different price ranges  
- Analyzed manufacturer contribution and pricing trends  
- Extracted high-value and low-cost products  
- Compared average pricing across manufacturers
