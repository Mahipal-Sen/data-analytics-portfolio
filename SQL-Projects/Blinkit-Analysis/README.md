#  Blinkit Grocery Data Analysis using SQL

##  Project Overview
This project focuses on analyzing grocery sales data for Blinkit using SQL.
The goal is to extract meaningful insights related to product performance, customer preferences, and outlet trends to support data-driven decision-making.

##  Tools & Technologies
- MySQL
- SQL (Structured Query Language)

##  Dataset Description
The dataset contains sales information for grocery items across multiple outlets with features such as:
- Item Identifier, Item Type, Item Fat Content  
- Item Weight, Item Visibility, Item MRP  
- Outlet Size, Location Type, Outlet Type  
- Item Outlet Sales  

##  Key Analysis Performed
- Data retrieval and filtering using SELECT, WHERE, BETWEEN, LIKE  
- Aggregations using COUNT, SUM, AVG, MAX, MIN  
- Grouping and sorting using GROUP BY and ORDER BY  
- Analysis of product categories and fat content  
- Outlet-based analysis (size, type, location)  
- Sales trend analysis across different attributes  
- Identification of high-performing products and outlets  

##  Key Insights
- Identified top-performing product categories  
- Analyzed impact of outlet type and location on sales  
- Compared sales based on fat content (Low Fat vs Regular)  
- Evaluated pricing and visibility impact on sales
