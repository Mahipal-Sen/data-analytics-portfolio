#  Olympics Data Analysis using Tableau

##  Project Overview
This project analyzes Olympic Games data (1896–2016) to uncover insights related to athlete performance, medal distribution, country participation, and sports trends. The objective is to use data visualization to identify patterns and support data-driven insights for large-scale sporting events.

##  Tools & Technologies
- Tableau (Data Visualization)
- Data Blending
- Exploratory Data Analysis (EDA)

##  Dataset Description
The project uses two datasets:
1. Olympic Athletes Data (athlete details, events, medals)
2. Olympic Countries Data (country and region details)

These datasets are connected using the common column **NOC (National Olympic Committee)**.

##  Key Analysis & Visualizations
- Stacked Bar Chart: Medal distribution (Gold, Silver, Bronze) by top 10 countries  
- Histograms: Age distribution of athletes across sports and medal categories  
- Donut Chart: Gender distribution with country filter  
- Side-by-Side Bar Chart: Medal distribution by gender  
- Line Charts:
  - Athlete participation over years  
  - Country participation trends  
  - Teams participation (Summer vs Winter)  
- Treemap: Top sports based on total medals  
- Filled Map: Medal distribution by country  
- KPI Cards:
  - Total Medals (Gold, Silver, Bronze)  
  - Total Athletes  
  - Total Sports  

##  Dashboard Features
- Interactive dashboards with filters  
- Country-level and gender-based analysis  
- Clear comparison of trends over time  
- Multiple visualizations integrated in a single view  

##  Key Insights
- Identified top-performing countries and sports  
- Analyzed gender participation trends in Olympics  
- Observed growth in athlete and country participation over time  
- Explored age distribution patterns across different sports  

##  Conclusion
This project demonstrates how Tableau can be used to analyze large-scale historical datasets and generate insights into global sporting events like the Olympics.
