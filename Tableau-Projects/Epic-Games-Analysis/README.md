# 🎮 Gaming Data Analysis using Tableau (Epic Games Case Study)

## 📊 Project Overview
This project focuses on analyzing the "Video_Games" dataset to uncover key insights into the gaming industry. The objective is to understand sales trends, player preferences, platform performance, and the relationship between critic scores and user engagement to support data-driven decision-making for an online gaming platform like Epic Games.

## 🛠 Tools & Technologies
- Tableau (Data Visualization)
- Data Analysis
- Exploratory Data Analysis (EDA)

## 📁 Dataset Description
The dataset includes information such as:
- Game Name, Platform, Genre, Publisher, Developer  
- Regional Sales (NA, EU, JP, Other) and Global Sales  
- Critic Score, User Score, Ratings, Release Year  

## 📈 Key Analysis & Visualizations
- Highlight Table: Genre-wise sales across regions  
- Bar Chart: Top 5 platforms by global sales  
- Lollipop Chart: Top 5 games by global sales  
- Funnel Chart: Top 10 games by user count  
- Area Chart: Sales trend over time  
- Combo Chart: Critic Score vs User Score by genre  
- Tree Map: Top publishers by number of games  
- Packed Bubble Chart: Top developers by contributions  
- Side-by-Side Bar Chart: Regional sales by age rating  
- Pie Chart: Distribution of games by rating  
- KPI Cards: Total sales, regional sales, average scores  

## 📊 Dashboard Features
- Interactive and user-friendly dashboard  
- Dynamic filters for better data exploration  
- Clear visualization of trends, patterns, and comparisons  

## 🔍 Key Insights
- Identified top-performing platforms and game genres  
- Analyzed regional sales differences across markets  
- Observed relationship between critic scores and user ratings  
- Highlighted leading publishers and developers in the industry  

## 🚀 Conclusion
This project demonstrates the use of Tableau to transform raw gaming data into meaningful insights, helping businesses like Epic Games make strategic decisions based on data.
