#  Blinkit Grocery Sales Analysis using Tableau

##  Project Overview
This project analyzes grocery sales data for Blinkit, an online grocery platform, to uncover insights related to sales performance, customer behavior, and outlet efficiency. The goal is to use data visualization to support data-driven decision-making and improve business operations.

##  Tools & Technologies
- Tableau (Data Visualization)
- Data Analysis
- Exploratory Data Analysis (EDA)

##  Dataset Description
The dataset contains information about grocery products and outlet performance, including:
- Product details: Item Type, Fat Content, MRP, Weight  
- Outlet details: Size, Location Type, Establishment Year, Outlet Type  
- Sales data: Item Outlet Sales (target variable)  

##  Key Analysis & Visualizations
- Top 10 selling items based on total sales  
- Sales comparison across different outlet types  
- Sales performance across outlet locations  
- Donut chart for distribution of item fat content  
- Line chart showing impact of outlet age on sales  
- Top-selling item types by total sales  
- Visualization of average MRP by item type  
- KPI Cards: Total Sales, Average MRP, Number of Items  
- Additional custom visualization for deeper insights  

##  Dashboard Features
- Interactive dashboard with year filter  
- Consistent Blinkit theme and formatting  
- User-friendly layout for easy interpretation  
- Integrated multiple visualizations in one view  

##  Key Insights
- Identified top-performing products and item categories  
- Analyzed how outlet type and location impact sales  
- Observed relationship between outlet age and sales performance  
- Highlighted customer preferences based on product features  

##  Conclusion
This project demonstrates how Tableau can be used to analyze retail sales data, generate insights, and support business decisions to improve efficiency and customer satisfaction.
